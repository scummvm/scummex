MCMP

MCMP contains compression info for WAV files and iMUS blocks in Grim Fandango. In this way, it corresponds to the COMP block of The Dig and CMI.

"MCMP" dword Block identifier
nEntries word Number of entries (BE)
 
loop nEntries
 nCodec byte Codec Number - index of the codec name in the list below.
 dwDecompSize dword Decompressed size (BE)
 dwCompSize dword Compressed size (BE)
end of loop
 
nCodecs word Number of codecs in list (LE)
 
loop nCodecs
 szCodecName 5 * char Codec Name - null terminated string. In Grim Fandango this is either NULL (for uncompressed) or VIMA (for the Grim Fandango music compression)
end of loop
