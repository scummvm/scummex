CLUT

CLUT contains palettes (i.e. color lookup tables) in the same format as APAL and NPAL. This format is simple: 256 3-byte values, one for each palette index.

"CLUT" dword Block identifier
dwSize dword Size in bytes (BE)
 
loop 256
 nRed byte Red intensity
 nGreen byte Green intensity
 nBlue byte Blue intensity
end of loop
